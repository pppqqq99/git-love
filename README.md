## Tables
확장된 Markdown 문법에서는 표를 만들 수 있다.

    | Syntax      | Description |
    | ----------- | ----------- |
    | Header      | Title       |
    | Paragraph   | Text        |
위와 같이 |와 세 개 이상의-를 활용하여 표를 만들 수 있다. 위 코드를 탭 없이 나타내면 아래와 같이 표로 표현된다.  

| Syntax      | Description |
| ----------- | ----------- |
| Header      | Title       |
| Paragraph   | Text        |
